# Repositório de Projetos - Bootcamp DIO

Este repositório contém projetos desenvolvidos durante o meu tempo no bootcamp da VIVO TELEFÔNICA pela [DIO](https://digitalinnovation.one/).

Cada projeto inclui arquivos de código-fonte e documentação necessária para entender e executar as soluções desenvolvidas.

## Tecnologias

Utilizei a linguagem Python

## Instruções de Uso

1. Clone o repositório.
2. Instale as dependências: `npm install` ou `pip install -r requirements.txt`.
3. Execute o projeto conforme as instruções nos respectivos diretórios ou arquivos README dos projetos.
